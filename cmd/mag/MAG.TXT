・X1　LSX-Dodgers専用外部コマンド

●MAG		鮪(MAG)形式のグラフィックのロード/セーブ

●書式

 MAG ファイル名[.MAG][スイッチ]

ロード系のスイッチ

	/C		画面クリア
	/R		くわの式変換を行なわないでそのまま表示

セーブ系のスイッチ

	/S		セーブ
	/H		200ライン・セーブ
	/U"user"	制作者名をセット

●解説

・まぐろ方面で開発された鮪(MAG)形式のグラフィックをロード、セーブする為のプログラムです。

・ファイル名 で指定されたMAG形式のグラフィックのロード、セーブを行ないます。

・ロードの際にはワイルドカードを用いて連続して見ることができます。
ロード中に[ESC]を押すと中止して次の画像に移ります。[SHIFT]+[BREAK]で中止します。
[BREAK]で一時停止します。16(8)ラインごとにキーを受付ます。

・拡張子を省略すると.MAGとなります。

・256色には対応していません、スキップします。

・扱える画像の大きさは横1024ドットまでで左640ドット分表示できます。

・縦はフリーエリアにもよりますが2000ライン程度は表示できます。画面を越える部分はキー待ちと
なり、何かキーを押すとスクロールしながら表示します。

・セーブ系のスイッチを使用するとセーブします。

・ロードの際は画面モードに応じて200/400ラインでロードを切り換えます。

・ノンターボや低解像度(15kHz)、インタレース・モードの場合は400ラインの画像なら、くわの式を用い
て200ラインで表示します。
・高解像度(24kHz)や縦長インタレース・モードの場合には400ラインで表示します。

・/Rスイッチを指定しなければアナログ16色の画像はくわの式を用いて8色に減色処理をしなが
ら表示します。

・/Uスイッチで指定しなければ制作者名は「>謎<」となります。

・MAKI形式には対応していません。

・X1turboZシリーズには320×200(4096色)で表示するMAGA.COMもあります。

●例

・B:ドライブのMAG画像をすべて表示する。

MAG B:* /C

・FILE.MAGでセーブ(400ライン)

MAG FILE /S

・FILE.MAGでセーブ(200ライン)

MAG FILE /H

・制作者名を Lovers にしてFILE.MAGでセーブ(400ライン)

MAG FILE /U"Lovers"
